  2. 十六进制 RGB 值：#RRGGBB <=> #{红}{绿}{蓝}
    RR: 十六进制的红色值，值域 HEX：00 ~ FF，（对应 DEC: 0 ~ 255）
    GG: 十六进制的绿色值，值域 HEX：00 ~ FF，（对应 DEC: 0 ~ 255）
    BB: 十六进制的蓝色值，值域 HEX：00 ~ FF，（对应 DEC: 0 ~ 255）

    熟练辨识：十六进制与对应的十进制数
    如：(HEX) 32 (DEC) 50、(HEX) 64 (DEC) 100

    简写：如果红绿蓝三色的十六进制数两两相同，可都简写为一位
    例如：#00CC33 可以简写为 #0C3，而 #335099 则不能简写，#3509 是错误的！

    安全色：在任意浏览器里显示的差异性最小的配色
    所有分色（红绿蓝）都使用：00 33 66 99 CC FF。
    熟悉相关数值：
                #0066CC    rgb(  0, 102, 204)
            #3399FF    rgb( 51, 153, 255)

  3. 十进制 RGB 值：rgb(Red, Green, Blue)
    Red 红色十进制值，值域 DEC：0 ~ 255
    Green 红色十进制值，值域 DEC：0 ~ 255
    Blue 红色十进制值，值域 DEC：0 ~ 255
  4. HSL 值：hsl(H, S%, L%)
    • H（色调）Hue，值域：0 ~ 360（0或360 红，120 绿，240 蓝）
    • S（饱和度）Saturation，值域：0.0% ~ 100.0%
    • L（亮度）Lightness，值域：0.0% ~ 100.0%
